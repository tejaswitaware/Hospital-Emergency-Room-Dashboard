# Hospital Emergency Room Dashboard 

## Overview

This project is an **Excel-based dashboard** that analyzes hospital emergency room data. It helps visualize patient statistics, waiting time, admission status, and departmental referrals through interactive charts and KPIs.

## Features

* Monthly patient analysis
* Average waiting time tracking
* Patient satisfaction score
* Admission vs Not Admitted analysis
* Gender-wise patient distribution
* Age group analysis
* Department referral statistics

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Dashboard Design

## Purpose

The goal of this project is to **analyze emergency room performance and patient flow** using data visualization to support better hospital decision-making.
